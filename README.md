# CustomAdview

without google ads 

Add it in your root build.gradle at the end of repositories:


	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
  
  
  Add the dependency
  
  	dependencies {
	        implementation 'com.github.Harshil258:CustomAdview:1.0.5.3'
	}
